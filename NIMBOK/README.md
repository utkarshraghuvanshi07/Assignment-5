# 🌿 Lombok - Holistic Health Landing Page

A clean and modern landing page built using **HTML5** and **CSS3**. This project recreates a wellness-inspired website with multiple sections, responsive layout principles, and a minimalist design.

> 🚧 Learning Project

---

## ✨ Features

- Hero landing section
- Clean navigation bar
- Latest Articles section
- About section
- Newsletter subscription section
- Footer with social media icons
- Hover effects
- Lazy loading for images
- Semantic HTML structure
- Smooth scrolling

---

## 🧠 Concepts Practiced

- Semantic HTML5
- CSS Flexbox
- Background Images
- Typography
- Hover Effects

- Responsive Layout Basics
